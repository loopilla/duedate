# DueDate

Foobar is a Python library for dealing with word pluralization.

## Installation

Clone git repository
In the project directory run: npm install

```bash
npm install
```

## Usage
tests:
```bash
    npm run test
```

build:
```bash
    npm run build
```

run:
```bash
    npm run start
```
